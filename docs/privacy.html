La presente informativa descrive in modo trasparente come vengono trattati i dati personali degli utenti che utilizzano la demo gratuita o la versione completa di "Barman PRO" (di seguito "Servizio"), in conformità al Regolamento UE 2016/679 ("GDPR") e al D.Lgs. 196/2003 e successive modifiche.

1. Dati trattati — demo gratuita

La demo pubblica disponibile su raccasamuele.github.io/barman-pro/ è progettata per minimizzare la raccolta di dati personali.

  • Nessun account, nessuna registrazione. La demo non richiede creazione di profilo né raccolta diretta di dati personali.
  • Nessun cookie, nessun tracker. Il Servizio non utilizza cookie, pixel di tracciamento, analytics o strumenti di profilazione.
  • localStorage tecnico: il Servizio salva nel browser dell'Utente (memoria locale del dispositivo) le seguenti informazioni, esclusivamente per garantire il funzionamento dell'app:
    • parametri dell'evento inseriti (numero ospiti, cocktail/persona, ecc.);
    • cocktail e shot selezionati per la serata;
    • lingua dell'interfaccia preferita.
    Tali dati restano sul dispositivo dell'Utente, non vengono trasmessi a server esterni e possono essere cancellati in qualsiasi momento dalle impostazioni del browser oppure tramite il pulsante "Azzera tutto" all'interno dell'app.
  • Log del server di hosting: il provider di hosting della demo (GitHub Pages) registra automaticamente l'indirizzo IP e lo user-agent del visitatore a fini di sicurezza e funzionamento del servizio. GitHub è titolare autonomo di tali log; per dettagli vedere la privacy policy di GitHub.

2. Dati trattati — versione completa (a pagamento)

L'acquisto e l'attivazione della versione completa comportano i seguenti trattamenti.

  • Dati di acquisto: nome, indirizzo email, paese, eventuale codice fiscale o partita IVA. Tali dati sono raccolti dal Merchant of Record Lemon Squeezy (Lemon Squeezy LLC, Delaware, USA), che agisce in qualità di titolare autonomo del trattamento per le finalità di fatturazione, antifrode e adempimenti fiscali globali. Per dettagli si rimanda alla privacy policy di Lemon Squeezy.
  • License Key: a seguito dell'acquisto, l'Utente riceve via email una chiave di licenza alfanumerica. Tale chiave viene salvata nel localStorage del browser dell'Utente per consentire l'apertura dell'app senza reinserimento ad ogni avvio.
  • Validazione antifrode della License Key: al primo inserimento della chiave e periodicamente (circa una volta a settimana), il Servizio effettua una chiamata HTTPS verso i server API di Lemon Squeezy per verificare la validità e la non revoca della chiave. In tale chiamata viene trasmesso il solo dato necessario (la License Key) e, automaticamente, l'indirizzo IP del dispositivo (informazione tecnica inevitabile in qualsiasi richiesta HTTP). L'IP non viene salvato localmente dall'app né comunicato a terzi diversi da Lemon Squeezy.

This Privacy Policy transparently describes how personal data of users of the free demo or the full version of "Barman PRO" (the "Service") are processed, in compliance with EU Regulation 2016/679 ("GDPR") and applicable national laws.

1. Data processed — free demo

The public demo available at raccasamuele.github.io/barman-pro/ is designed to minimize personal data collection.

  • No account, no registration. The demo does not require profile creation or direct collection of personal data.
  • No cookies, no trackers. The Service does not use cookies, tracking pixels, analytics, or profiling tools.
  • Technical localStorage: the Service stores the following information in the User's browser (local device memory), solely to ensure the app functions correctly:
    • event parameters entered (number of guests, cocktails per person, etc.);
    • cocktails and shots selected for the night;
    • preferred interface language.
    Such data remains on the User's device, is not transmitted to external servers, and can be deleted at any time from browser settings or via the "Reset all" button within the app.
  • Hosting server logs: the demo's hosting provider (GitHub Pages) automatically logs the visitor's IP address and user-agent for security and operational purposes. GitHub is the autonomous controller of such logs; for details see the GitHub privacy policy.

2. Data processed — full version (paid)

Purchasing and activating the full version involves the following processing.

  • Purchase data: name, email, country, possibly tax ID. Such data is collected by the Merchant of Record Lemon Squeezy (Lemon Squeezy LLC, Delaware, USA), acting as autonomous data controller for billing, anti-fraud, and global tax compliance purposes. See Lemon Squeezy's privacy policy for details.
  • License Key: after purchase, the User receives an alphanumeric license key by email. This key is stored in the User's browser localStorage to allow the app to open without re-entry on every launch.
  • Anti-fraud License Key validation: upon first key entry and periodically (approximately once a week), the Service issues an HTTPS call to Lemon Squeezy's API servers to verify the validity and non-revocation of the key. Only the strictly necessary data (the License Key) is transmitted in such calls, along with the device's IP address (technical information inevitable in any HTTP request). The IP is not stored locally by the app nor shared with parties other than Lemon Squeezy.
